⚠️ PHP Konfigurationsfehler

Die Website kann nicht angezeigt werden, da PHP-Dateien nicht korrekt ausgeführt werden.

Problem:
Der Webserver behandelt PHP-Dateien als Downloads statt sie auszuführen. Dies deutet auf eine fehlerhafte PHP-Handler-Konfiguration hin.
Lösungsschritte für All-Inkl:
1. In KAS einloggen
2. Domain firmenpro.de auswählen
3. PHP-Version auf 8.4 prüfen
4. .htaccess-Override aktivieren
5. FastCGI/CGI-Modus kontrollieren

Technische Details:

  • Server: All-Inkl (KAS)
  • PHP-Version: 8.4 (nicht aktiv)
  • Handler: application/x-httpd-php84
  • Status: PHP wird als Download ausgeliefert
